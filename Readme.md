
1. Create empty folder

2. Type following git command
            'git init'

3. Create Readme.md file

4. Add some coll stuff in that file

5. Type following command to stage the file for commit
            'git add Readme.md'

6. Commit the changes to your local repo by typing following command
            'git commit -m "commit massage"'

7. Push the changes to your remote repo ‘git push https://github.com/rajenderpasham/CDTraining.git’
